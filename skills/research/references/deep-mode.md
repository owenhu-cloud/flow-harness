# research 深档（deep mode）

> 高风险/不可逆决策、结论将被长期依赖、单源生态、安全/合规时启用。浅档不必走本文件。
> 借鉴 Anthropic 多代理研究系统（orchestrator-worker + 独立 CitationAgent）与 deep-research 管线的「证伪回环 + 引用红线」。

## 管线（七步，Critique 可回环）

1. **Scope** —— 把问题拆成可独立检索的子问题；写下「什么样的证据能改变本次决策」。
2. **Plan** —— 每个子问题派一个只读子代理（`Task`），给定：单一目标、产出格式（结论+引用+源类型+置信度）、完成边界、反注入声明。
3. **Retrieve** —— 子代理抓**一手源**（官方文档/源码/规范/一手数据），不靠记忆、不靠二手转述。
4. **Triangulate（三角核验）** —— 每条关键断言至少 3 个**独立**源交叉印证；来源互为转载不算独立。
5. **Synthesize** —— 主线程只收蒸馏结论，合成权衡表/结论集，标每条置信度。
6. **Critique（红队证伪回环）** —— 用下方人格主动证伪；任一关键结论被动摇 → 回到第 3 步补查，不带未核验结论出门。
7. **Package** —— 落盘 `docs/flow/<change>/research.md`（结论综述）+ `docs/flow/<change>/sources.md`（每条断言→源清单，抗上下文压缩）。

## 红队人格（Critique 阶段轮流戴上）

- **怀疑的实践者**：「这个结论在真实生产/边界条件下还成立吗？谁踩过坑？」
- **对抗审稿人**：「样本/版本/时间是否过期？有没有相反证据被忽略？引用是否真支持该断言？」
- **落地工程师**：「按这个结论动手，第一个会炸的地方是什么？前提条件齐了吗？」

## 引用红线（违反即不得据此拍板）

- **每条关键断言 ≥3 独立源**；不足则显式标 ⚠ 低置信，禁止作为决策依据。
- **可达性校验**：每条引用的 URL/DOI 必须真实可达且内容确实支撑该断言——禁编造引用、禁「我记得有篇论文说」。
- **源可信度分级**：一手（官方文档/源码/规范/标准）＞ 同行评议 ＞ 信誉机构博客 ＞ 个人博客转述。低级别源支撑高风险结论时必须升级核验。
- **落盘抗压缩**：`sources.md` 保留「断言→源→源类型→访问日期」，上下文被压缩后仍可复核，不靠记忆。

## 硬指标（深档达标线）

- 总源数 ≥ 10；每条关键断言独立源 ≥ 3。
- 每条关键结论都经过至少一轮红队证伪尝试并记录结果（成立/降置信/推翻）。
- 结论与 `sources.md` 一一对应，无悬空断言。

不达标 = 深档未完成，退回浅档结论并显式标注「未达深档强度」，不冒充已深核。

## 机器校验（sources.md 格式 + 校验脚本）

`sources.md` 用可被脚本解析的格式（`## ` 起一条断言块，每个 `- ` 行含一个 URL/doi 即计一个源）：

```
## <断言文本>
- <源类型> | https://...
- <源类型> | doi:10.xxxx/...
- <源类型> | https://...
```

落盘后跑校验器（gate-able，纯 POSIX sh，非 0 即不达标）。脚本随本技能分发，路径 = **技能加载时显示的 "Base directory" + `/references/verify-citations.sh`**（cwd 是用户项目根，别用相对路径找它）；`sources.md` 路径相对用户项目根：

```
sh "<本技能 base directory>/references/verify-citations.sh" docs/flow/<change>/sources.md
```

它校验：① 每条断言独立源数 ≥ 阈值（默认 3，`MIN_SOURCES=` 可调）；② 每个 URL/DOI 真实可达（curl HEAD，2xx/3xx）。源数不足或引用不可达即 `exit 2` 列出问题——**补足独立源或剔除不可达引用，禁编造引用充数**。离线/测试可用 `FLOW_URL_CHECK` 注入自定义可达性检查器（见 `verify-citations.test.sh`）。

## 收尾硬门（HARD GATE：深档不跑校验器、不绿，不算完成）

> 深档收尾**必须**跑 `verify-citations.sh` 且 **exit 0**，并在完成声明里**贴出它的新鲜输出**。
> 校验未跑 / 未过 = 深档未完成，禁止据未校验结论拍板——与 Flow「完成必带证据」红线同性质。
> 自检一句：「校验器今轮跑过吗？绿了吗？输出贴了吗？」任一为否 → 留在深档，别交回 plan。
